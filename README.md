# 長野市 バドミントンコート 空き検索

> **免責・利用目的について**
>
> 本ツールは、**個人の利便性向上のみを目的**として作成したものです。
> 商業利用・営利目的は一切ありません。
> 特定の個人・団体・サービスを攻撃・妨害・誹謗中傷する意図はまったくありません。
> データは長野市 施設案内予約システム（公開情報）を参照しているだけであり、
> サーバーに過負荷をかけないよう取得結果を 24 時間キャッシュしています。
> 問題がある場合はリポジトリの Issue にてご連絡ください。即座に対応します。

---

長野市の体育館の空きコマを一覧表示するローカル専用ツールです。
[長野市 施設案内予約システム](https://city.nagano.nagano.machikagi-remote.jp) の公開情報を取得しており、**ログイン不要**で利用できます。

距離の基準は **長野駅** です。

---

## 機能

- 日付・利用時間帯・最大距離でフィルタリング
- 体育館のみ対象（`tag=1`）
- 有料 / 無料タブで結果を分類
- 長野駅からの距離順で表示
- 施設・部屋情報は 24 時間キャッシュ（サーバー負荷軽減）

## セットアップ

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## 起動

```bash
.venv/bin/streamlit run finder.py
```

ブラウザで `http://localhost:8501` が開きます。

## 技術スタック

| 項目 | 内容 |
| --- | --- |
| 言語 | Python 3.10+ |
| UI | Streamlit |
| 施設検索 | machikagi-remote.jp（公開HTML） |
| 空き枠取得 | `/rooms/{id}/reservation_events.json` |
| ジオコーディング | 国土地理院 API（APIキー不要） |
| 距離計算 | Haversine 公式 |
